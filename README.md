## 📌 과정 소개

| 항목 | 내용 |
|:---:|:---|
| 과정 | PLC 기초 제어 실습 |
| 대상 | 자동화 / 전기 / 제어 입문 학습자 |
| 목표 | PLC의 기본 개념을 이해하고 간단한 제어 프로그램을 작성할 수 있다 |

---

## 🎯 강의목표

- PLC의 기본 구조와 동작 원리를 이해할 수 있다.
- 입력과 출력의 개념을 알고 제어 흐름을 구성할 수 있다.
- 타이머, 카운터를 활용한 기초 제어를 구현할 수 있다.
- 실제 자동화 설비에서 사용하는 기본 제어 로직을 작성할 수 있다.

---

## 🗓️ 강의 구성 (강 단위)

| 강차 | 강의 및 실습 내용 | 핵심 키워드 |
|:---:|:---|:---|
| 1강 | PLC 개념 및 구조 이해 | PLC, 입력, 출력 |
| 2강 | PLC 동작 원리 (스캔 사이클) | Scan Cycle |
| 3강 | A접점 / B접점 이해 | NO, NC |
| 4강 | 코일 및 기본 회로 구성 | Coil |
| 5강 | 자기유지 회로 실습 | START, STOP |
| 6강 | 인터록(상호배제) 회로 | Interlock |
| 7강 | 타이머 기초 | Timer |
| 8강 | 카운터 기초 | Counter |
| 9강 | 램프 / 부저 제어 | 출력제어 |
| 10강 | 모터 제어 기초 | 운전, 정지 |
| 11강 | 센서 입력 처리 | Sensor |
| 12강 | 응용 회로 구성 | 자동화 로직 |
| 13강 | 종합 실습 1 | 제어 통합 |
| 14강 | 종합 실습 2 | 프로젝트 |

## 🛠️ 실습 예시

- 버튼 → 램프 점등
- 스타트 / 스톱 회로
- 타이머 지연 동작
- 카운터 개수 제어
- 센서 감지 → 출력 동작

---

## 📚 학습 방향

- 입력 → 판단 → 출력 흐름 이해
- 실제 설비 기준으로 생각하기
- 작은 회로라도 끝까지 완성하기
- 반복 실습을 통한 제어 감각 확보
